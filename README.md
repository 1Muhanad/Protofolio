E-Commerce Website
Overview

A modern and responsive E-Commerce web application built with React. The project allows users to browse products, view product details, add items to the cart, and enjoy a smooth shopping experience.

Features
Responsive design for all devices.
Product listing page.
Product details page.
Shopping cart functionality.
Search and filtering.
Modern UI and smooth user experience.
State management for cart and products.
Technologies Used
HTML5
CSS3
React.js
Libraries
React Router DOM (for routing)
React Icons
